# hospital-appointment-management
Web application for hospital appointment management, including user registration, login, admin dashboard, hospital listing, appointment requests, approval workflow and real-time chat. Built with PHP, MySQL, HTML, CSS, and JavaScript.


A complete web application that manages hospital appointments with patient registration, login, real-time chat, hospital listing, admin validation, and appointment workflows. Built during a Hackathon to demonstrate practical PHP/MySQL full-stack development.

---

##  Features
-  User registration & authentication  
-  Hospital listing and search  
-  Appointment creation & management  
-  Admin dashboard for approval/refusal  
-  Real-time chat between patients and hospitals  
-  Secure PDO-based database connection  
-  Clean UI using HTML, CSS, and JavaScript  

---

##  Technologies Used
- **PHP (PDO)**
- **MySQL**
- **HTML5 / CSS3**
- **JavaScript**
- **XAMPP / Apache**

---
##Import database

- Open phpMyAdmin

- Create a new database (example: santeo)

- Import file: id2912909_santeo.sql

## Configure database connection

Modify connection.php:

$host = "localhost";
$dbname = "santeo";
$username = "root";
$password = "";

## Run the project

Visit in browser:

http://localhost/hospital-appointment-management/
